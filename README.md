# 使い方
1. Python3.11(pipも)と、screenをインストール
2. 
```
python3.11 -m pip install MiPA
cp .env.example .env
```
3. .envファイルの中身を編集する(これをしないと動きません)
4. ./start.shを実行
5. 楽しむ！
